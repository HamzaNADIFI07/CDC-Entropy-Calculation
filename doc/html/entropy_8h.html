void count_occurrences ( FILE * file,
int counts[] )

Counts the occurrences of all the characters in the file given as the first parameter (the FILE pointer was obtained with the fopen function) and puts all the counts in the counts array.


Parameters
fileA file opened in read mode
countsThe counts array must be of size 256, and all cells must initially be at 0.
Postcondition
the counts array will contain the number of occurrences of each symbol (ie. counts[i] contains the number of occurrences of symbol i).
